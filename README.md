# 🌱 EnviroChart Dashboard

A modern environmental data dashboard built with React, TypeScript, and TailwindCSS.

## 🚀 Features

- Interactive charts (CO₂, Energy, Water)
- Dynamic filtering (3 / 6 / 12 months)
- Dark mode support 🌙
- Responsive dashboard layout
- Clean and reusable component structure

## 🛠 Tech Stack

- React + TypeScript
- Vite
- TailwindCSS
- Recharts
